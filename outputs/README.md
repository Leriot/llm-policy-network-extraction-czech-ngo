# Outputs Directory

Final thesis outputs are organized here.

- `figures/` - network maps, distributions, diagnostic plots, and validation-effect figures.
- `tables/` - centrality tables, thesis tables, and exported statistical summaries.
- `reports/` - rendered or text reports used in the thesis workflow.
- `model_results/ergm/` - ERGM summaries, fit metrics, predictive comparisons, GOF plots, and MCMC diagnostics.

Most files here are generated outputs. They are retained in GitHub for convenience because they correspond to final thesis tables, figures, or model diagnostics and are small enough for regular Git. The Zenodo deposit may also include a copy of these outputs for a complete archival record.
